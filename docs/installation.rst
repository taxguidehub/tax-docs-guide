Installation Guide
==================

This section explains how to install tax software on your device.

System Requirements
-------------------

- Windows or Mac computer
- Stable internet connection
- Minimum 4GB RAM

Installation Steps
------------------

1. Download the software from the official source.
2. Open the installer file.
3. Follow on-screen instructions.
4. Complete the setup process.

Notes
-----

Ensure your system meets all requirements before installation.
